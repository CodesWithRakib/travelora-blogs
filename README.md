# goshomik – Travel Blog

A modern, performant travel blog built with **Next.js**, **TypeScript**, **Tailwind CSS**, **shadcn/ui**, and **Sanity** as the headless CMS.  
Inspired by slow travel and storytelling, this project showcases a clean, minimal design with a focus on readability and user experience.

---

## ✨ Features

- ⚡ **Blazing fast** – Static generation with Incremental Static Regeneration (ISR)
- 📱 **Fully responsive** – Optimized for all devices
- 🎨 **Modern UI** – Clean, minimal design using Tailwind CSS and shadcn/ui components
- 📝 **Rich content** – Portable Text from Sanity with custom image and link handling
- 🔍 **SEO ready** – Semantic HTML, meta tags, JSON‑LD structured data (Article, Blog, Person)
- 🌍 **Dynamic sitemap** – Automatically generated sitemap for all blog posts
- 📂 **Infinite scroll / pagination** – Load more posts on the blog listing
- ♿ **Accessible** – Focus management, skip links, ARIA labels
